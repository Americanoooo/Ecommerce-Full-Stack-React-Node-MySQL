#  TechStore Ecommerce

🇧🇷 Português

Projeto de ecommerce desenvolvido com React focado em aprendizado prático de frontend, responsividade e gerenciamento de estado.

O projeto simula uma loja virtual moderna com catálogo de produtos, página individual de produto e carrinho de compras funcional.

---

## Funcionalidades

- Página inicial com listagem de produtos
- Página individual de produto
- Carrinho de compras
- Aumento e remoção de quantidade
- Cálculo automático de valores
- Rotas dinâmicas
- Busca de produtos
- Layout responsivo
- Componentes reutilizáveis

---

##  Tecnologias

- React
- React Router DOM
- TailwindCSS
- JavaScript

---

##  Aprendizados

Durante o desenvolvimento desse projeto foram praticados conceitos como:

- gerenciamento de estado
- componentização
- manipulação de arrays e objetos
- responsividade
- renderização dinâmica
- rotas no React
- lógica de carrinho de compras

Atualmente o projeto está evoluindo para uma versão full stack utilizando:

- Node.js
- Express
- MySQL
- APIs REST
